## Here is how to make the script works

# Install

* Copy protocol.zip to Desktop .
* Open The Terminal .
* Type the following commands :

```
    - [X] cd Desktop
    - [X] unzip protocol.zip
    - [X] cd protocol
    - [X] chmod +x protocol
    - [X] ./protocol  or  bash protocol or sh ./protocol
```
